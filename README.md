# copilot-commercant

## Overview
`copilot-commercant` is a Next.js application designed to provide a seamless e-commerce experience. This project leverages the latest features of Next.js, including server-side rendering, API routes, and TypeScript support.

## Features
- **Dynamic Routing**: Easily create dynamic routes for products and categories.
- **API Integration**: Built-in API routes for handling data fetching and manipulation.
- **TypeScript Support**: Strongly typed components and functions for better development experience.
- **Responsive Design**: Mobile-friendly layout with global styles.

## Getting Started

### Prerequisites
- Node.js (version 14 or later)
- npm (version 6 or later) or Yarn

### Installation
1. Clone the repository:
   ```
   git clone https://github.com/yourusername/copilot-commercant.git
   ```
2. Navigate to the project directory:
   ```
   cd copilot-commercant
   ```
3. Install dependencies:
   ```
   npm install
   ```
   or
   ```
   yarn install
   ```

### Running the Development Server
To start the development server, run:
```
npm run dev
```
or
```
yarn dev
```
Open your browser and navigate to `http://localhost:3000` to see the application in action.

### Building for Production
To create an optimized production build, run:
```
npm run build
```
or
```
yarn build
```

### Running in Production
To start the production server, run:
```
npm start
```
or
```
yarn start
```

## Folder Structure
- `src/app`: Contains the main application files including layout, pages, and API routes.
- `src/components`: Reusable components like Header and Footer.
- `src/lib`: Utility functions for API calls and database interactions.
- `src/styles`: Global and variable CSS files for styling.
- `src/types`: TypeScript type definitions.
